# odin-recipes

A simple, multi-page website showcasing various recipes.

## 🚀 Live Demo:

[Link to live demo.](https://thepaladindev.github.io/odin-recipes/)

## Tech Stack:

- HTML5 (Pure HTML, no CSS/JS yet)
- git

## Getting Started

To view this project locally:

1. Clone the repository.
   ```bash
   git clone git@github.com:ThePaladinDev/odin-recipes.git
   ```
2. Navigate into the directory.
   ```bash
   cd odin-recipes
   ```
3. Start a http server in the project directory in one of these ways:<br>
   - Using VS Code Live Server extension.
   - Using npx. (You might have to install 'npm' as well as the 'serve' package)

   ```bash
   npx serve .
   ```

   - Using Python. (You might have to install Python)

   ```bash
   python -m http.server
   ```

4. Once the http server starts, it will print the url it is listening on in your terminal.<br>
   The url is something like `http://localhost:PORT/`.<br>
   The default port for:
   - VS Code Live Server is `5500`.
   - npx is `3000`.
   - python http server is `8000`.

5. Enter the url in your browser.

6. Tap `Ctrl+C` in the terminal to stop the server (npx, and python) once you're done testing the project.

## Features:

- Main Menu: The home page listing all available recipes.
- Recipe Pages: Individual pages with info on ingredients and preparation steps.
- Navigation: Internal linking allows users to jump between menu and recipes easily.
